# Error Handling

All Guarantee functions return `Result<T, SdkError>`. The error type covers every failure mode in the SDK.

## `SdkError`

```rust
pub enum SdkError {
    /// Attestation hardware or interface unavailable
    #[error("Attestation unavailable: {0}")]
    AttestationUnavailable(String),

    /// Failed to read DCAP quote from /dev/attestation
    #[error("Failed to read quote: {0}")]
    QuoteReadFailed(String),

    /// Ed25519 key generation failure
    #[error("Key generation failed: {0}")]
    KeyGenerationFailed(String),

    /// Ed25519 signing failure
    #[error("Signing failed: {0}")]
    SigningFailed(String),

    /// File I/O error (sealing, unsealing)
    #[error("I/O error: {0}")]
    IoError(#[from] std::io::Error),

    /// EnclaveAttestor used before initialization
    #[error("Attestor not initialized")]
    NotInitialized,

    /// Sealing or unsealing failure
    #[error("Seal error: {0}")]
    SealError(String),

    /// AES-256-GCM encryption/decryption failure
    #[error("Crypto error: {0}")]
    CryptoError(String),

    /// RA-TLS certificate or server failure
    #[error("RA-TLS error: {0}")]
    RaTlsError(String),

    /// X.509 certificate parsing failure
    #[error("Certificate error: {0}")]
    CertificateError(String),

    /// TLS configuration failure
    #[error("TLS error: {0}")]
    TlsError(String),
}
```

## Handling errors

Guarantee uses `thiserror` for error types, so they work naturally with the `?` operator:

```rust
use guarantee::errors::SdkError;

fn my_function() -> Result<(), SdkError> {
    let state = TeeState::initialize(std::path::Path::new("./sealed"))?;
    state.seal(std::path::Path::new("./sealed"))?;
    Ok(())
}
```

## In axum handlers

Convert to HTTP responses:

```rust
#[attest]
async fn handler(
    Extension(state): Extension<Arc<RwLock<TeeState>>>,
) -> impl IntoResponse {
    let s = state.read().await;

    match s.decrypt_user_record(&encrypted) {
        Ok(user) => Json(serde_json::json!(user)).into_response(),
        Err(SdkError::CryptoError(msg)) => (
            StatusCode::INTERNAL_SERVER_ERROR,
            Json(serde_json::json!({
                "error": { "code": "DECRYPTION_FAILED", "message": msg }
            })),
        ).into_response(),
        Err(e) => (
            StatusCode::INTERNAL_SERVER_ERROR,
            Json(serde_json::json!({
                "error": { "code": "INTERNAL_ERROR", "message": e.to_string() }
            })),
        ).into_response(),
    }
}
```
